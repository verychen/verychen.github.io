android studio安装
1 安装jdk1.8
安装目录：
C:\Program Files\Java\jdk1.8.0_51\
设置JAVA系统环境变量
1 JAVA_HOME C:\Program Files\Java\jdk1.8.0_51\
2 CLASSPATH：.;%JAVA_HOME%/lib/tools.jar;%JAVA_HOME%/lib/dt.jar
3 path目录首部添加：;%JAVA_HOME%/bin;

ps:注意安装和系统对应的版本；

安装
1 没有更新sdk导致无法进入android studio

问题解决：
如果是第一安装完之后出现这个界面，在这个目录下
C:\Program Files\Android\Android Studio\bin\idea.properties

#———————————————————————
idea.xdebug.key=-Xdebug

#———————————————————————–

Change to ‘enabled’ if you want to receive instant visual notifications

about fatal errors that happen to an IDE or plugins installed.

#———————————————————————–
idea.fatal.error.notification=disabled

#第一次打开报错
disable.android.first.run=true
